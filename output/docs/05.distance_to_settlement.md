# Preparing Distance to Settlement Covariate for Multiseason Occupancy Analysis

## Objectives:

To calculate the distance of the site covariates from the settlement
areas in order to understand the influence of anthropogenic activities
in the species occupancy

## Data Source:

Settlement data were obtained from the Global Human Settlement Layer
(GHSL): - product: GHS-BUILT-S - Epochs: 2015, 2018, 2020 - Resolution:
100 m (2015, 2020), 10 m (2018) - Source:
<https://ghsl.jrc.ec.europa.eu/>

## Workflow Overview

-   Download GHSL settlement raster data
-   Reproject raster to UTM Zone 44 N
-   Clip raster to a buffered study area
-   Create binary settlement raster
-   Generating Euclidean distance raster
-   Extracting mean distance values for each sampling grid

## Detailed Workflow

### Step 1: Download settlement layer

The GHSL built-up surface data for 2015, 2018 and 2020 was downloaded
and stored in 'data/raw/ghsl_set' ![Visualizing raw ghsl layer in
QGIS](D:/R_projects/multiseason_occupancy/output/figures/Raw%20ghsl%20settlement%20layer.png)

### Step 2: Reprojected Raster

The raster layers were reprojected to EPSG:32644-WGS84/ UTM Zone 44 N
using Raster - Projections - Warp(Reproject) and the resolution was
standardized to 100 m.

This ensures that all distance calculations are performed in meters

### Step 3: Prepared buffered study area

The grid layers from the study_area were dissolved using vector -
Geoprocessing tools - Dissolve Output: dissolved_SA

Then a 10 km buffer was created around the dissolved study_area using
vector - Geoprocessing tools - Buffer Output: buffer_SA

This step minimizes the edge effects during the proximity analysis
![Dissolved study
area](D:/R_projects/multiseason_occupancy/output/figures/dissolve_sa.png)

### Step 4: Clip the buffered study area

The settlement raster layer was clipped to the buffered area for easy
analysis Raster - Extraction - Clip raster by mask layer Output:
clip_set_2015.tif, clip_set_2018.tif, clip_set_2020.tif

### Step 5: Created binary raster layer

A binary raster layer was created from the settlement layer where 1
indicated presence of settlement area in the pixel and 0 indicated
absence of settlement area in the pixel Raster - Raster calculator -
Expression: ("[set_2015\@1](mailto:set_2015@1){.email}" \> 0 ) \* 1

This step was repeated for 2018 and 2020 layer. Output:
binary_set_2015.tif, binary_set_2018.tif, binary_set_2020.tif ![Binary
settlement
layer](D:/R_projects/multiseason_occupancy/output/figures/binary_settlement%20layer_creation.png)

### Step 6: Converted NoData values to 0

No data values were converted to 0 to ensure continuous distance
propagation during proximity analysis. Raster - Raster calculator -
Expression:
IF(is_null("[binary_2015\@1](mailto:binary_2015@1){.email}"), 0,
"[binary_2015\@1](mailto:binary_2015@1){.email}")

Output: fix_set_2015.tif, fix_set_2018.tif, fix_set_2020.tif

### Step 7: Calculated distance to settlement in raster

The distance to settlement was calculated using proximity(Raster
Distance) tool where, Raster - Analysis - Proximity(Raster Distance)
Input = fix_set_2015.tif List of target pixels = 1 Distance units =
georeferenced coordinates Output file = Dist_set_2015.tif ![Distance to
settlement
raster](D:/R_projects/multiseason_occupancy/output/maps/distance_to_settlement.png)

### Step 7: Calculated distance value for each site

The distance of the sample sites from the nearest settlement area was
calculated using zonal statistics where, Input = sample_sites.gpkg
Raster layer = Dist_set_2015.tif Statistics to Calculate = Mean Output:
Dist_set_2015.gpkg

### Step 8: Join the distance to settlement of each year

The values of distance to nearest settlement for the sample sites in
each year was compiled in one attribute table using Dist_set_2015.gpkg -
properties - join Input = Dist_set_2018.gpkg Join field = siteID Target
field = siteID ![Joining the
layers](D:/R_projects/multiseason_occupancy/output/figures/Joining%20the%20layers.png)

### Step 9: Export the information as a csv file

The data was exported as the Dist_settlement.csv in the
output/Dist_settlement.csv to be used as a site covariate for
multiseason occupancy analysis.

## Final Outputs

-   dist_settlement_cov.csv

### Ecological Relevance

Distance to settlement was used as an indicator of human disturbance and
accessibility. Areas closer to settlements are generally associated with
greater anthropogenic pressure while remote areas may provide more
suitable habitat for disturbance sensitive wildlife species.

# Preparation of Observation (Survey) covariates for Multiseason Occupancy Analysis

## Objectives

To prepare survey covariate matrix to identify the variables
that influence the detection probability in multi-season occupancy
analysis of a species.

## Data Source

-   Camera trap data

## Software Used

-   R programming 4.5.3

## Workflow Overview

-   Convert the date in the csv files from character to date format
-   Create all possible occasion for every site and year
-   Calculate start and end date of each occasion
-   Create effort matrix
-   Create year matrix

## Detailed Workflow

### Step 1: Convert the deploy date and end date from character to date format

The dates from csv file were converted from character to date format
using ymd() function of lubridate for further analysis.

`data <- data %>% mutate( deploy_date = ymd(deploy_date), ended_date = ymd(ended_date)`

### Step 2: Create all possible occasion for every site and year

Every possible occasion combinations for site year i.e. one row per site
\* occasion was created using rowwise() and list() functions.

`effort_long <- data %>% rowwise() %>% mutate( occasion = list(1:n_occasions) ) %>% unnest(occasion)`

### Step 3: Calculate start and end date of each occasion

The start and end date of each occasion was calculated with 5
consecutive days as one occasion in order to ensure the dates fall into
correct occasion category. The active camera days was also calculated.

`effort_long <- effort_long %>%`

`mutate( occ_start = deploy_date + (occasion - 1) * 5, #calculate start of each occasion# occ_end = pmin(occ_start + 4, ended_date), #calculate ending day of each occasion# effort = as.numeric(occ_end - occ_start) + 1 #calculate active camera days)`

### Step 4: Create effort matrix

With the active camera days calculated, a wide format effort matrix was
formed with same dimension as encounter hisotry

`effort_matrix <- effort_long %>% pivot_wider( names_from = occasion, values_from = effort, names_prefix = "occ_" )`

### Step 5: Create Year Matrix

The year from the date was extracted and was used to create a wide
format year matrix with same dimension as encounter history

`year_matrix <- effort_long %>% mutate(year_cov = year) %>% select(site_year, occasion, year_cov) %>% pivot_wider( names_from = occasion, values_from = year_cov, names_prefix = "occ_" )`

### Step 6: Export the observation covariate matrices

The matrices were exported as a csv file to be used as observation covariates in multiseason occupancy analysis.

`write.csv(effort_matrix, "D:/R_projects/multiseason_occupancy/output/effort_matrix.csv") 
write.csv(year_matrix, "D:/R_projects/multiseason_occupancy/output/year_matrix.csv")`

## Outputs

-   effort_matrix.csv
-   year_matrix.csv
